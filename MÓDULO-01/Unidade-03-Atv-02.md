# Atividade Avaliativa – Testes Não Funcionais

## Sistema Proposto: Loja Virtual

O sistema permite que o usuário realize cadastro, login, pesquisa de produtos, compras online e acompanhamento de pedidos.

## Checklist de Testes Não Funcionais

| Categoria | O que será verificado | Risco associado |
|-----------|------------------------|-----------------|
| **Performance** | Tempo de carregamento da página inicial. | Lentidão pode fazer o usuário desistir de acessar o sistema. |
| **Performance** | Desempenho do sistema com muitos usuários simultâneos. | Quedas, travamentos ou indisponibilidade do sistema. |
| **Performance** | Tempo de resposta na finalização da compra. | Atrasos podem causar abandono da compra. |
| **Segurança** | Validação do login e autenticação dos usuários. | Acesso indevido a contas de clientes. |
| **Segurança** | Proteção dos dados pessoais e de pagamento. | Vazamento de informações confidenciais. |
| **Segurança** | Controle de permissões para administradores e clientes. | Usuários podem acessar funções não autorizadas. |
| **Usabilidade** | Facilidade de navegação entre as páginas. | Dificuldade de uso e abandono do sistema. |
| **Usabilidade** | Clareza das mensagens de erro e confirmação. | Usuários podem não entender como corrigir problemas. |
| **Usabilidade** | Facilidade para concluir uma compra. | Processo confuso pode reduzir as vendas. |
| **Compatibilidade** | Funcionamento nos navegadores Chrome, Edge e Firefox. | O sistema pode apresentar erros em alguns navegadores. |
| **Compatibilidade** | Funcionamento em dispositivos móveis e computadores. | Parte dos usuários pode não conseguir utilizar o sistema. |
| **Compatibilidade** | Adaptação a diferentes tamanhos de tela (responsividade). | Layout quebrado ou dificuldade de navegação. |

---

# Conclusão

Os testes não funcionais são essenciais para garantir a qualidade do sistema além das funcionalidades. Eles avaliam desempenho, segurança, facilidade de uso e compatibilidade, reduzindo riscos como lentidão, falhas de segurança, dificuldades de navegação e problemas em diferentes dispositivos e navegadores.
