## 💻 Exemplos de Blocos de Código

### 📌 Comandos Git

```bash
# Clonar um repositório
git clone https://github.com/seuusuario/seuprojeto.git

# Verificar o status
git status

# Adicionar arquivos
git add .

# Criar um commit
git commit -m "Primeiro commit"

# Enviar para o GitHub
git push origin main
```
